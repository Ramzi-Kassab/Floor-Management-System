Upload Guidelines
Supported File Types
  • Documents: PDF, Word, Excel, PowerPoint
  • Images: JPG, PNG, GIF, SVG, WebP
  • Video: MP4, AVI, MOV, WebM
  • Audio: MP3, WAV
  • CAD: DWG, DXF
  • Archives: ZIP, RAR, 7Z
File Size Limits

Maximum file size: 50 MB
For larger files, contact IT support.

Naming Conventions

Use descriptive titles that include:

  • Document purpose
  • Department/Area
  • Version if applicable

Version Control

Use semantic versioning (1.0, 1.1, 2.0) to track document revisions. Major changes = increment first number. Minor changes = increment second number.
