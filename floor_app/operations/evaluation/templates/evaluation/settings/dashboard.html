Settings Overview

The Evaluation module settings allow you to configure:

  • Cutter Codes: Define evaluation codes with actions and color indicators
  • Feature Codes: Configure geometry types and feature identifiers
  • Bit Sections: Define sections of the bit for structured evaluation
  • Bit Types: Configure different bit type classifications
  • Instruction Templates: Create reusable technical instruction templates
  • Requirement Templates: Build validation checklists for quality control
